GDD Printers

To install and use the majority of the GDD printers, you will need to install the GDD Mosyle profile onto your MacBook. The link to the profile download will be provided at the beginning of the semester via your student email.

Once the profile is installed it should take a couple minutes for the printer drivers to download and install. Check to see if they're installed under system preferences and printers and scanners.

C1423 Konica AccurioPrint C4065​

Click here for Konica Tutorial

The Konica AccurioPrint C4065 is GDD's main printer. It's located in Building C room C1423.

To print to the Konica from your Mac:

  • Click File -> print or press ⌘ + P on your keyboard to bring up the print menu.
  • Select Konica from the printer menu
  • Choose "Fiery Features" from the drop down, if you need to set paper sizes, duplexing, etc..
  • Click Print

To release your print job, walk over to the printer:

  • On the printers screen, tap Controller located at the bottom
    • Then tap Waiting Jobs
    • And then tap Held Jobs
    • In the "log in" screen, Select Operator and tap Log In
  • Find your print job in the list and hit print!

Printing on Cardstock​

When printing on cardstock you Must Only use the bypass tray.

  • Load your cardstock in the bypass Tray
  • From your print menu, make sure to select "bypass tray" before you hit print.
  • Go to the printer and release the print as normal.

Prints to the Konica, even test prints, will contribute to your 75 prints.

C1420 Xerox VersaLink C7030 Printer​

Click here for Versalink Tutorial

The main GDD pod studio's printer is the Xerox VersaLink C7030, located in C1420

note

To install and use any FDD printer you will need to download and install the GDD profile. The link will be emailed to your student email address.

Once the profile is installed, under system settings, you will see the Xerox Versalink 7030 installed in your printers list.

To print using the Xerox C7030:

  • Click File -> print or press ⌘ + P on your keyboard to bring up the print menu.
  • Select C1420 Versalink from the printer list
  • Click Print

To access the printers special features

  • Click File -> print or press ⌘ + P on your keyboard to bring up the print menu.
  • Select C1420 Versalink from the printer list
  • Click Copies & Pages and select Xerox Features
  • Navigate the menu and select your Feature
  • Click Print
important

It's recommended you install the printer by installing the Mosyle profile, but if you cannot install Mosyle you can find the printer driver at: https://www.support.xerox.com/en-us/product/versalink-b7025-b7030-b7035/downloads?platform=macOS10_15&language=en

Also, you can find the Xerox ICC profiles here: https://www.support.xerox.com/en-gb/product/versalink-c7020-c7025-c7030/content/144388

Printing on Cardstock​

The paper attributes for the tray you want to print from must match the paper attributes in the print driver in order for the document to print properly.

At the Printer:

  1. Load the Cardstock in the Bypass Tray
  2. In the Paper Settings (Bypass) screen
  3. Confirm the following:
  • Size: Letter 8.5 x11” paper.
  • Type: Cardstock (or other media type)
  • Color: If other than White
  • Confirm: Upper right corner of screen

At the computer: Configure settings and print from the application:

  1. Select File > Print > Select Printer Properties
  2. Verify the “Paper” settings > Size “Letter (8.5x11”)”
  3. Click dropdown arrow
  4. Select “Other Type”
  5. Select “your Cardstock type”
  6. (Optional) Click “Output Destination” dropdown
  7. Select Center Tray.
  8. Press OK, > Print > Your document is now sent to the Xerox MFP.

Scanning​

  1. To scan to your email address, press the scan-to button on the control panel.
  2. Load your page.
  3. type in your email address.
    • Scroll down for more options such as file type, DPI, etc..
  4. press the scan button at the top.
  5. check your email.
note

You will find additional how-to print guides on posters near the printers

How to Print​

  1. Download and install a 3D slicing software such as PRUSA Slicer https://www.prusa3d.com/page/prusaslicer_424/
  2. Load your .STL file in the slicing software and configure it as neccecary.
  3. Slice it, and save the new file to an SD card.
  4. Insert the SD card into the 3D printers SD slot.
  5. Load the Filament into the 3D printer.
  6. Turn the knob on the display until you see your file then press the button.

Creating an .STL file for printing​

You can create or modify an existing .STL file using a 3D animation software such as Blender https://www.blender.org/download/

If you are starting from scratch, build your model then export as .STL file. If you want to modify an existing .STL file go to file - import - STL.

Loading Filament​

You will need to load the filament before use, and unload it and store when you are finished.

To Load Filament:

  1. Place spool onto printer's spool holder.
  2. turn the knob on the printer until it says "load filament" Press the button.
  3. wait until the nossle heats up and lets you know its ready to load.
  4. take the end of the filament and start feeding it into the extruder until it grabs it and outputs some melted filament.
  5. select OK when prompted.

To Unload Filamanet:

  1. turn the knob until the dislay says "unload filament" press the knob
  2. gently pull filament up and out of the extruder.
  3. Respool the end of the filament, put it back in its zip lock bag, and store it.
