MacBook Pro Option:â€‹

tip

If you want the best battery life go with the 16" model with the M1 Pro chip. Go with the 16" model as well if you do not have an external monitor to plug it into. If money is tight and you want better portability or plan to use it primarily docked to a monitor, go with the 14" model.

14" or 16" 2021 MacBook Pro

  • M1 Pro Chip
  • 8 Core CPU
  • 14 Core GPU
  • 32GB Unified Memory (16gb minimum)
  • 512GB SSD Storage
  • AppleCare warranty
  • Pro Apps Bundle NOT Required

Starts at $3,372.00 + taxes (educational price)
