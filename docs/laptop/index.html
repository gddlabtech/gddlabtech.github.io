GDD Laptop Requirements

Required Equipment and Software Checklist:#

  • 16" MacBook Pro
  • High-Speed Stable Internet: For remote learning classes
  • External Hard Drive:
    LaCie 1 TB USB-C or alternative
    Samsung Portable SSD Faster Version
  • SD Memory Card: 32 GB
  • Apple AirPods or Alternative Headset with Mic

Optional Equipment:#

  • Canon or Nikon Digital SLR camera, or Sony mirrorless camera

Options are preconfigured at Simply Computing

MacBook Pro Option:#

16" 2021 MacBook Pro with touch bar and Thunderbolt 3 (USB-C) ports

  • 2.6GHz 6-core 9th-generation Intel Core i7 processor, Turbo Boost up to 4.5GHz
  • 32GB 2666MHz DDR4 memory
  • 512GB SSD
  • Radeon 5300m with 4GB GDDR6 memory
  • AppleCare warranty
  • Pro Apps Bundle NOT Required

$3,534.00 + taxes (educational price)

MacBook Pro Higher-End Option:#

16" 2021 New MacBook Pro with touch bar and Thunderbolt 3 (USB-C) ports

  • 2.3GHz 8-core 9th-generation Intel Core i7 processor, Turbo Boost up to 4.8GHz
  • 32GB 2666MHz DDR4 memory
  • 1TB SSD
  • Radeon 5500m with 8GB GDDR5 memory
  • AppleCare warranty
  • Pro Apps Bundle NOT Required

$4,168.00 + taxes (educational price) Adjust options accordingly

MacBook Pro Absolute Minimum Requirement:#

  • 2018 Retina MacBook Pro or newer
  • 2 GHz Quad-core Intel i7 Processor or better
  • MUST have dedicated video card of 2GB or better
  • 16GB memory
  • 512GB Solid State drive
important

Please confirm and connect with a GDD Lab Technician before buying a used computer.
