GDD Laptop Requirements

Required Equipment and Software Checklist:​

  • 2024 MacBook Pro

  • High-Speed Stable Internet: For remote learning classes

  • SD Memory Card: minimum 32 GB

Optional Equipment:​

  • Canon or Nikon Digital SLR camera, or Sony mirrorless camera
  • External Hard Drive:
    LaCie 1 TB USB-C or alternative
    Samsung Portable SSD Faster Version

MacBook Pro Option:​

tip

If you want the best battery life go with the 16" model with the M4 Pro chip. Go with the 16" model as well if you do not have an external monitor to plug it into. If money is tight and you want better portability or plan to use it primarily docked to a monitor, go with the 14" model.

14" or 16" 2024 MacBook Pro

  • M4 Pro Chip
  • 12 Core CPU
  • 16 Core GPU
  • 24GB Unified Memory (16gb minimum)
  • 512GB SSD Storage
  • AppleCare warranty
  • Pro Apps Bundle NOT Required

Starts at $3,166 + taxes (educational price) Adjust options accordingly

MacBook Pro Higher-End Option:​

Recommended for those wishing to pursue video, animation, or motion graphics within your Graphic Design career.​

14" or 16" 2024 MacBook Pro

  • M4 Max Chip
  • 14 Core CPU
  • 32 Core GPU
  • 36GB Unified Memory
  • 1TB SSD Storage
  • AppleCare warranty
  • Pro Apps Bundle NOT Required

Starts at $4,801 + taxes (educational price) Adjust options accordingly

MacBook Pro Absolute Minimum Requirement:​

  • 2021 M1 MacBook Pro or newer
  • 8 Core CPU
  • 14 Core GPU
  • 16GB memory
  • 512GB Solid State drive
important

Please confirm and connect with a GDD Lab Technician before buying a used computer.

Are you on a student loan?​

Equipment costs for this program are eligible for funding. Contact Financial Aid and Awards for more information. https://www.ufv.ca/fineaid/contact-us/
