Off Campus File Access‚Äã

note

Using the GDD Cloud you can only remotely access files that you can access on the Abbotsford GDD File Server. If you are not on campus and need files that are on the Mission GDD File Server you must connect to Mission GDD WiFi.

The GDD Cloud is a server that we in Graphic and Digital Design host for student, faculty, and class use. The GDD Cloud is distributed via NextCloud software that allows you to access the Graphic and Digital Design server while studying remotely. If you have ever used a cloud based file sharing service such as Google Drive, or Dropbox, you should feel right at home in the GDD Cloud as most things will work similarly between them!

Web browser requirements‚Äã

For the best experience with the GDD Cloud web interface, we recommend that you use the latest and supported version of a browser from this list:

  • Microsoft Internet Explorer
  • Microsoft Edge
  • Mozilla Firefox
  • Google Chrome/Chromium
  • Apple Safari

Navigating the main user interface‚Äã

By default, the GDD Cloud Web interface opens to your Dashboard or Files page:

In Files you can add, remove, and share files, and make changes based on the access privileges set by you (if you are administering the server) or by your server administrator.

The GDD Cloud user interface contains the following fields and functions:

  • Apps Selection Menu (1): Located in the upper left corner, you‚Äôll find all your apps which are available on your instance of GDD Cloud. Clicking on an apps icon will redirect you to the app.
  • Apps Information field (2): Located in the left sidebar, this provides filters and tasks associated with your selected app. For example, when you are using the Files apps you have a special set of filters for quickly finding your files, such as files that have been shared with you, and files that you have shared with others. You‚Äôll see different items for other apps.
  • Application View (3): The main central field in the GDD Cloud user interface. This field displays the contents or user features of your selected app.
  • Navigation Bar (4): Located over the main viewing window (the Application View), this bar provides a type of breadcrumbs navigation that enables you to migrate to higher levels of the folder hierarchy up to the root level (home).
  • New button (5): Located in the Navigation Bar, the New button enables you to create new files, new folders, or upload files.
note

You can also drag and drop files from your file manager into the Files Application View to upload them to your instance. Currently, the only Web browsers that support drag-and-drop folders are Chrome and Chromium.

  • Search field (6): Click on the magnifier in the upper right hand corner of to search for files.
  • Contacts Menu (7): Gives you an overview about your contacts and users on your server. Dependent on the given details and available apps, you can directly start a video =- call with them or send emails.
  • Grid view button (8). This looks like four little squares, which toggles the grid view for folders and files.
  • Settings menu (9): Please do not change any settings here.

Accessing your files using the GDD Cloud Web interface‚Äã

If you have ever used a file sharing service such as Dropbox or Google Drive, a lot of the interface here should be pretty straightforward!

Tags and Comments‚Äã

We do ask that you refrain from using the tags system available to keep the interface clear. But we encourage you to use comments, as this can be a great way to comment on your fellow students work.

Use the Details view to add and read comments on any file or folder. Comments are visible to everyone who has access to the file:

Overflow menu‚Äã

The Overflow menu (three dots) displays file details, and allows you to rename, download, or delete files:

Creating or uploading files and directories‚Äã

Upload or create new files or folders directly in a GDD Cloud folder by clicking on the New button in the Files app:

  • Up arrow Upload files from your computer into GDD Cloud. You can also upload files by dragging and dropping them from your file manager.
  • Text file Creates a new text file and adds the file to your current folder.
  • Folder Creates a new folder in the current folder.

Selecting files or folders‚Äã

You can select one or more files or folders by clicking on their checkboxes. To select all files in the current directory, click on the checkbox located at the top of the files listing.

When you select multiple files, you can delete all of them, or download them as a ZIP file by using the Delete or Download buttons that appear at the top.

warning

DO NOT DELETE OTHER STUDENTS or INSTRUCTORS FILES OR FOLDERS!

Moving Files‚Äã

You can move files and folders by dragging and dropping them into any directory.
